# MCP IoT Server

Điều khiển thiết bị IoT (ThingsBoard) và giám sát telemetry.

## Tools

### ThingsBoard
- `set_attributes_light` - Điều khiển đèn (tranh A, B, C, D)
- `send_chart_telemetry` - Lấy dữ liệu telemetry & vẽ biểu đồ

## Cài đặt

```bash
pip install -r requirements.txt
```

## Sử dụng

```bash
python server.py
```
