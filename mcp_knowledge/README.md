# MCP Knowledge Server

Tìm kiếm thông tin trên Internet (Serper) và tra cứu kiến thức (Ragflow).

## Tools

### Search
- `search_internet` - Tìm kiếm thông tin trên Internet

### Knowledge
- `ragflow_query` - Tra cứu kiến thức từ các nguồn (Lily, Rùa Trạng Nguyên, Y tế...)

## Cài đặt

```bash
pip install -r requirements.txt
```

## Sử dụng

```bash
python server.py
```
