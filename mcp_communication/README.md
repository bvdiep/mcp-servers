# MCP Communication Server

Đọc email, dự báo thời tiết và lấy thời gian hiện tại.

## Tools

### Email
- `read_emails` - Đọc email gần đây từ nhiều hộp thư (Inbox, Notification, Newsletter)

### Weather
- `get_weather_forecast` - Dự báo thời tiết cho các thành phố

### Time
- `get_current_time` - Lấy thời gian hiện tại của một thành phố

## Cài đặt

```bash
pip install -r requirements.txt
```

## Sử dụng

```bash
python server.py
```
